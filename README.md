# RR Painters Website

A premium, modern, and responsive showcase website developed for **RR Painters (R. R Painting Services Limited)**, a professional painting company based in New Zealand.

The website is designed to showcase completed painting projects, highlight services, build customer trust, and provide an easy way for potential customers to request quotes.

Clone the repository:

Run these commands:

```bash
cd RR-painters-website

npm run dev
```
